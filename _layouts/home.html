// My Words

A part of who I am, a part of my very core, is to try more than one thing. Life is an experiment, this is how I live. And that is reflected also in how and where I write and publish online. I joined Twitter back in 2006. I was on Facebook in 2005. I had a blog on Xanga back in 2005 as-well. Those were my firsts. Since then, I've published on LinkedIn (🤢), WordPress.com, WordPress.org (oh so many self-hosted sites there), and more recently Micro.blog, Scribbles, omg.lol (especially my status), and most recently, as of February 22, 2026, Pika. Anywho. Keeping up with me can be a thing...and maybe that is changing now.

One thing I have come to learn through all of this is that I love to tinker. I love to write, I enjoy coding, I enjoy building from scratch, learning as I go, and trying to do random fun things with my sites.

A few years ago I started this website as a test. It is built using Jekyll, each page is a static page hosted on GitHub Pages, and it even has its own python app that I built to update my status. It is dead simple on the front-end. And it offered a great learning curve on the backend! I have built a lot into this site and now plan to use it as my online home, one I think I have been searching for for quite a while.

Anyways...I always tend to go on and on and on. If you want to learn more about this site you can read about that here. Otherwise, clicking/tapping the various buttons below will take you deeper down the rabbit hole.

Thanks for visiting and thanks for reading,
Aaron Aiken
